Users

HTTP Methods Path Description
GET / (Home Page)

Use the forward slash to see the Home Page

GET /Users

Use this HTTP method and endpoint to display/ see all the Users.
GET /User/:id

Use this HTTP method and endpoint to display/ see only one User by provided the id of that User.
POST /User/add

Use this HTTP method and endpoint to add a User.
PATCH /User/:id

Use the HTTP method and endpoint to edit a User's information.
DELETE /User/:id

Use this HTTP method and endpoint to delete a User .
